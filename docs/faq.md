# FAQ

## What is Boson?

Boson is an open-source, local-first coding workbench focused on direct AI operator access.

## How is this different from hosted AI editors?

Boson is built around user control: open codebase, flexible providers, and local-machine-first workflows.

## Can I use my own model provider?

Yes. The project direction is provider-agnostic integration through adapter interfaces.

## Can this run fully offline?

Core UI and local workbench behavior can run locally. AI features depend on selected providers and whether they are local or remote.

## Is Boson production-ready?

Boson is actively evolving. Core workbench flows are in place, while deeper operator integration and some parity features are still in progress.

See `CONTRIBUTING.md` for setup and contribution workflow.
